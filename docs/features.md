# 機能仕様

## AI サービス切り替え

画面左下のアイコンをクリックすることで AI サービスを順番に切り替えます。

| 状態 | アイコン | AI |
|------|---------|-----|
| 0 | claude-ai.png | Claude (Anthropic) |
| 1 | chatgpt-ai.png | ChatGPT (OpenAI) |
| 2 | gemini-ai.png | Gemini (Google) |

切り替えはローテーション形式 (0 → 1 → 2 → 0) で動作します。

## モデル切り替え

モデルアイコンをクリックすることで高性能モード / 経済モードを切り替えます。

| フラグ (`b`) | ChatGPT | Claude | Gemini |
|-------------|---------|--------|--------|
| `1` (高性能) | chatgpt-4o | claude-3-7-sonnet | gemini-2.0-flash |
| `0` (経済的) | gpt-4o-mini | claude-3-5-haiku | gemini-2.0-flash |

> デフォルトは高性能モード (`model = 1`)。

## メッセージ送信

テキストエリアにメッセージを入力して SEND ボタンまたは Enter キーで送信します。

### テキストコマンド

入力欄に以下のコマンドを入力することで特殊な処理を実行できます。

| コマンド | 動作 |
|---------|------|
| `/raw` | 会話履歴をマークダウン変換なしで表示 (`\n` → `<br>` のみ) |
| `/all` | 会話履歴をマークダウン HTML に変換して表示 |
| `/image <プロンプト>` | DALL-E 3 による画像生成。結果画像を表示 |

### 会話履歴の表示形式

`/all` および `/raw` で表示される形式:

```
user: <メッセージ内容>

assistant: <AIの回答>
--------------------

user: <次のメッセージ>
...
```

## 音声入力

マイクアイコンをクリックすることで音声認識を開始・停止します。

- 言語: 日本語 (`ja`)
- 認識した音声はリアルタイムでテキストエリアに反映
- 以下の**音声コマンド**で自動送信が可能:

| 音声コマンド | 動作 |
|-------------|------|
| 「～エンター。」 | 「エンター」を除いたテキストを送信 |
| 「～送信。」 | 「送信」を除いたテキストを送信 |
| 「～教えて。」 | そのままのテキストを送信 |

## 画像入力

AI に画像を添付して質問できます。

### クリップボードから貼り付け

テキストエリアにフォーカスした状態で画像を貼り付け (`Ctrl+V`) すると:
- 自動的に最大 512×512px にリサイズ
- Base64 データ URL に変換
- プレビューとして表示

### ファイルのドラッグ＆ドロップ

テキストファイルをアプリ画面にドラッグ＆ドロップすると、ファイルパスと内容がテキストエリアに挿入されます。

挿入形式:
```
/path/to/file.rs
```rust
// ファイルの内容
\```
```

- ディレクトリや存在しないファイルはスキップ
- 複数ファイルを一度にドロップ可能

## システムプロンプト

ドロップダウンからシステムプロンプトを選択することで AI の振る舞いを変更できます。

### `[system]` プロンプト (Rust 側で管理)

バックエンドの `system_messages` に設定され、AI の役割を定義します。

| 選択肢 | 内容 |
|--------|------|
| 厳格で正確な | 法律・規則に従った正確かつ厳格な回答 |
| 肯定的な | ユーザーのモチベーション向上に注力 |
| 批判的な | 建設的な批判的視点を提供 |

### `[user]` プロンプト (フロントエンド側で管理)

テキストエリアに直接プロンプトを挿入します。

| 選択肢 | 内容 |
|--------|------|
| 自己分析 | 会話履歴から自己分析を行う |
| まとめ | 会話を要約・#タグ付きで出力 |
| 引き継ぎ | 後任への引き継ぎプロンプトを作成 |
| クライアントメール用応答 | 簡潔・結論ファーストのメール文面に変換 |
| クライアントメール用配慮 | 相手を慮った配慮メール文面に変換 |

## 会話履歴の保存 (メモ)

削除アイコン (delete.png) をクリックすると:
1. 現在の会話履歴をファイルに保存 (`memo()`)
2. 会話履歴をリセット (`reset`)

### 保存先

```
~/Documents/.appdata/Talk with RustGPT/
```

### ファイル名

| 条件 | ファイル名 |
|------|-----------|
| 会話内に `#タグ` がある | `memo-YYYY-MM-DD_HH-MM-SS-タグ名.txt` |
| タグなし | `memo-YYYY-MM-DD_HH-MM-SS.txt` |

### 自動保存

ウィンドウを閉じたときにも自動的にメモが保存されます (`on_window_event` で `Destroyed` イベントを監視)。

## レスポンス表示

AI からの回答は以下の形式で表示されます:

```
<マークダウン→HTML変換されたAI回答>

Model: <使用モデル名>, Total token: <合計トークン数>, Elaps: <経過秒数>s
```

- マークダウンは GFM (GitHub Flavored Markdown) に対応
- 数式 (math_flow / math_text) に対応
- テーブル・タスクリストに対応
- コードブロックは highlight.js でシンタックスハイライト

## 質問テキストの表示

最後に送信した質問は画面上部に表示されます。

- 25文字以上の場合は末尾に `...` で省略
- クリックするとクリップボードにコピーされる

## 画像生成 (DALL-E 3)

`/image <プロンプト>` コマンドで DALL-E 3 による画像生成を実行します。

- サイズ: 1024×1024px (固定)
- 生成後はプロンプトと画像 URL を表示
- 会話履歴には追加されません
